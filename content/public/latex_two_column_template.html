摘要

1.1 基础内容测试

本文档是一个面向学术会议论文排版的 HTML 模板，目标是在浏览器中模拟常见 LaTeX 双栏论文版式。模板参考 PubCSS 使用 HTML 与 CSS 排版学术出版物的思路， 将页面尺寸、页边距、双栏正文、标题区、作者信息、表格、图题和子图等常用结构整理为可直接复用的基础样式。

模板同时考虑中文与英文论文写作场景，默认使用适合中英文混排的字体栈，并保留 A4 页面、 印刷样式、表格横线、图表标题和占位图等会议论文中常见的排版元素。使用者可以在此基础上替换正文、 图表和作者信息，快速生成接近 LaTeX 双栏会议格式的 HTML 预览或打印输出。

关键词：HTML 模板；LaTeX 风格；双栏排版；学术会议；中英文支持
